# Product Overview

## Sijilli (سجلي)

Sijilli is a social appointment management application focused on organizing meetings and managing invitations between individuals. The app emphasizes simplicity, clarity, and a seamless user experience with full support for both Arabic (RTL) and English (LTR).

### Core Features

- **Appointment Management**: Create, view, and manage appointments with privacy controls (public/private)
- **Invitation System**: Invite users to appointments with status tracking (pending, accepted, rejected, deleted)
- **Social Features**: Follow system, user profiles with customizable bios and social links
- **Notifications**: Real-time updates for invitation responses and appointment changes
- **Search**: Find users and public appointments
- **Offline-First**: Full functionality with local caching and background sync

### User Types

- **Admin**: Full control over app, content, and users
- **User**: Can create appointments, invite others, and follow accounts

### Key Design Principles

- Simplicity and ease of use are fundamental
- Card-based UI for organized content presentation
- Every UI element has a clear, defined purpose
- Smooth, pleasant user experience
- TikTok-style vertical centering for profile pages
- Cache-first approach for instant loading
